Fixed Issues:

  • Fixed issue CERT - DNS Registration and Auto discovery Vulnerability and DNS rebinding attack.
  • Upgraded OpenSSL package.
  • Fixed Session ID vulnerability.

Known issues:

  • AP in client mode scans third party APs configured with WPA2PSK / WPAPSK authentication less when client mode AP is configured with OPEN-NONE or OPEN-WEP authentication and vice versa in congested environment..

NOTES:

  1. The default IP address of the AP is 192.168.0.100
  2. Restoring config file which has AP name with hyphen at the starting and end will not set the AP hostname properly.
  3. AP name configured with hyphen in the old FW version after firmware upgrade to latest image will not display proper AP name.
  4. In the following scenarios, AP is expected to reboot automatically for the configuration to take effect.
    • Change of Country/Region 
    • After Firmware upgrade 
    • After Restore Configuration Settings 
    • Reset factory defaults 
    • Change the mode from AP to Client and vice versa 
  5. WAC120 can operate in either AP or Client mode at any given time but not as both.

Installation

Note:  Before you upgrade to 2.1.7, Upgrade to 2.0.5 release. Please visit the link to download the Firmware version mentioned http://support.netgear.com/product/WAC120

  1. Download the new software and save it to a convenient folder location.
    The Firmware File name after extracting is wac120_v2.1.7_firmware.tar

    Download Link: http://www.downloads.netgear.com/files/GDC/WAC120/WAC120_firmware_V2.1.7.zip
  2. Login to the access point web management GUI.
  3.  Select Maintenance > Upgrade > Firmware Upgrade.
  4. Click Browse and browse to the location of the software upgrade file that you just downloaded.
  5. Click Apply to initiate upgrade process. Note: The upgrade process typically takes at least 6 minutes when upgraded to V2.0.5. Any upgrade from V2.0.5 to other Firmware version will take approximately 3 minutes.

WARNING: Do NOT interrupt or power off the computer until the upgrade process is completed. Wait for the AP to reboot.
